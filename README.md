
## 🎴 Duel Quest

A cross-platform digital trading card game inspired by classics like Pokémon TCG and Digimon TCG, built using C++ and SDL2.

---

## 🧩 About the Project

**Duel Quest** is a desktop-based trading card game that aims to deliver a strategic and engaging gameplay experience through a custom-designed card system and game mechanics. The project focuses on creating a unique blend of classic TCG elements with original gameplay features.

The game is designed to run on multiple platforms, making it accessible to a wide range of users across modern desktop environments.

---

## 🎮 Key Features

* 🃏 Original card system inspired by popular TCGs
* 🧠 Strategy-driven gameplay mechanics
* 🖥️ Cross-platform support (Linux & Windows)
* 🎨 Custom-designed digital card assets and UI
* ⚙️ Lightweight and efficient implementation using C++

---

## 🛠️ Tech Stack

* **Language:** C++
* **Graphics & Rendering:** SDL2
* **Digital Asset Creation:** GIMP, Photoshop
